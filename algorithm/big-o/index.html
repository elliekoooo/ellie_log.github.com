Time Complexity

  • What the concept of asymptotic runtime, Big O time.

    • Electronic Transfer: The size of file is going up, The time to transfer file is going up. O(s).
    • Airplane Transfer: THe size of file is going up, The time is constant. O(1).
  • linear will at some point surpass constant
  • Best case / worst case / Expected case

e.g. quick sort?

best case: when if all elements are equal, the quick sort will traverse the array once. <O(N)> worst case: when if the pivot is repeatedly the biggest element in the array? => Our recursion doesn't devide the array in half, and recurse on each half <O(N^2)> expected case: Sometimes the pivot will be very low or very high, but it won't always happen. <O(NlogN)>
